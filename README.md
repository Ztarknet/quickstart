# Ztarknet quickstart

## Installation

Get the latest Snfoundry (recommended way is via asdf): https://foundry-rs.github.io/starknet-foundry/getting-started/installation.html

## Create and deploy your account

```
make account-create
```

Go to the [faucet](https://github.com) and top up your account address.

```
make account-deploy
```

Verify you got the funds:

```
make account-balance
```
